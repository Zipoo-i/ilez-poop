## Быстрый Запуск

1. **Установите заранее Python и затем зависимости, открыв консоль в корневой папке проекта**
```
cd backend
pip install -r requirements.txt
```
2. **Запустите сервер**
```
python app.py
```
![alt text](image.png)
3. **Откройте приложение в браузере**

Локально: http://localhost:5000

С другого устройства если подключены к одной сети: http://[IPv4адрес-вашего-устройства-где-запущен-сервер]:5000
